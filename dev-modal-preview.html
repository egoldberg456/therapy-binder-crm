Outreach modal preview

Uses the same outreach-modal.js as the extension (single source of truth). The right column uses mock spreadsheet rows; the lower section uses mock Google Tasks. In Gmail, rows load via GET_OUTREACH_SHEET_ROWS and tasks via GET_OPEN_GOOGLE_TASKS.

Optional: run python3 -m http.server 8765 in this folder and open http://localhost:8765/dev-modal-preview.html if you prefer a URL over file://.
